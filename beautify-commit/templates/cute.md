# 可爱风格 (Cute Style)

## 风格描述

使用可爱的二次元风格 emoji 和颜文字，让 commit 记录充满活力和趣味。

## 格式规则

- 以合适的 emoji 开头
- 使用活泼可爱的语气
- 可以添加颜文字表情
- 保持专业性的同时增添趣味

## 推荐 Emoji

- ✨ (sparkles) - 新功能
- 🔧 (wrench) - 修复
- 📝 (memo) - 文档
- 🎨 (art) - 样式/UI
- ⚡ (zap) - 性能优化
- 🌸 (cherry blossom) - 美化
- 💫 (dizzy) - 重构
- 🎀 (ribbon) - 装饰性改动
- 🌟 (glowing star) - 重要更新
- 🎪 (circus tent) - 测试

## 常用颜文字

- (๑•̀ㅂ•́)و✧ - 加油
- (｡･ω･｡)ﾉ♡ - 开心
- ✧*｡٩(ˊᗜˋ*)و✧*｡ - 兴奋
- ₍₍ ◝(●˙꒳˙●)◜ ₎₎ - 快乐
- (｡♥‿♥｡) - 喜欢

```
✨ 哇！新增了超棒的登录功能呢~ (๑•̀ㅂ•́)و✧
🔧 修好了购物车的小bug啦！(｡･ω･｡)ﾉ♡
📝 完善了API文档呢~ ✧*｡٩(ˊᗜˋ*)و✧*｡
🌸 让界面变得更可爱了呢~ (｡♥‿♥｡)
⚡ 性能提升！现在快得飞起~ ₍₍ ◝(●˙꒳˙●)◜ ₎₎
💫 重构了认证模块，代码更清晰啦~ ✨
🎨 调整了配色方案，看起来更舒服了~ (๑•̀ㅂ•́)و✧
```
